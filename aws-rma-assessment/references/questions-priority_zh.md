# RMA 问题优先级分类

## 优先级定义

- **P0 - 关键**：直接影响系统可用性、RTO/RPO和客户体验，必须回答
- **P1 - 重要**：影响系统韧性和恢复能力，强烈建议回答
- **P2 - 建议**：最佳实践和持续改进，建议回答
- **P3 - 可选**：成熟度提升和组织文化，可选回答

包含 P0 + P1 优先级的所有问题

- **定期回顾**：建议季度简约评估，年度完整评估
